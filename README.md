# hackea
Hackea al Bullying es una iniciativa inspirada en el poder de la comunidad y la tecnología para enfrentar el acoso escolar. Buscamos informar, concienciar y brindar herramientas de apoyo a las víctimas, promoviendo el respeto, la empatía y la inclusión. Juntos transformamos el silencio en acción y construimos espacios más seguros.
